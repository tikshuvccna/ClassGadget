# ClassGadget - אולפן שידור מתקדם למרצים ומורים 🎓🎬

ClassGadget היא תוכנה חכמה שנועדה לשדרג את חוויית הלמידה מרחוק (בזום ובפלטפורמות נוספות). המערכת פותחה כ"אולר שוויצרי" למורים, ומאפשרת שליטה חיה בזמן שיעור ללא צורך בהגדרות מורכבות.

## ✨ מה חדש בגרסה האחרונה?
* **מצב "רואה ואינו נראה" (Stealth):** התוכנה הופכת לשקופה עבור תוכנות שיתוף מסך כמו Zoom, כך שהתלמידים לא רואים את חלון השליטה של המורה!
* **אינטגרציה חלקה לחלוטין עם ZoomIt:** התוכנה מגדירה את ZoomIt אוטומטית מאחורי הקלעים (ללא צורך בהגדרות ידניות), ומונעת התנגשויות של קיצורי מקלדת במחשב.
* **מנגנון חיתוך מסך מובנה (DPI Aware):** כלי חיתוך שקוף וידידותי המאפשר שמירה מהירה של אזורים מהמסך ישירות לתיקייה, עם תמיכה מלאה במסכים ברזולוציה גבוהה.

## 🛠️ פיצ'רים מרכזיים
* **פופאפים והנפשות:** הקפצת אנימציות (GIF) על גבי המסך בלחיצת כפתור או דרך קיצורי מקלדת נבחרים (Ctrl+Alt+1 עד 8).
* **כלי מסך מתקדמים:** הקלטת מסך וידאו, זום חי, וצייר על גבי המסך.
* **אפקטים קוליים (SFX) ומוזיקת רקע:** הפעלת מחיאות כפיים, תופים, שעון מעורר ומוזיקת המתנה.
* **כלים אינטראקטיביים:** גלגל שמות (מייבא אוטומטית שמות מהקובץ `students.txt` או `students.xlsx`), סקרים נגררים, ומבזק חדשות.
* **משחקי כיתה:** משחק 'איש תלוי' בעברית מלאה (כולל אותיות סופיות), אתגרי חשבון מתקדמים ומשחק מילה מבולגנת (מוסתר כוכביות).
* **מחולל פרומפטים מובנה (AI):** תבניות מוכנות מראש ליצירת חומרי לימוד וקוד בעזרת בינה מלאכותית.

## 🚀 איך משתמשים?
1. הורידו את קובץ ה-ZIP המלא מתוך לשונית ה-[Releases](../../releases) וחלצו אותו לתיקייה במחשב שלכם.
2. ודאו שהתיקיות (`gifs`, `sounds`, `recordings`) וקובץ ה-`ZoomIt.exe` נמצאים באותה תיקייה יחד עם קובץ ההפעלה `ClassGadget.exe`.
3. הפעילו את התוכנה (אין צורך בהתקנה), בחרו את המסך שאותו אתם משתפים בזום, והתחילו ללמד!

> **טיפ חשוב:** כל צילומי המסך והקלטות הוידאו שתיצרו דרך התוכנה יישמרו אוטומטית בתוך תיקיית ה-`recordings` עם חותמת תאריך ושעה.

## 💻 למפתחים (בנייה מתוך קוד המקור)
אם ברצונכם להריץ את הקוד ישירות או לארוז מחדש:
1. שכפלו את הריפוזיטורי (Clone).
2. ודאו שיש לכם קובץ `ZoomIt.exe` ותמונה בשם `icon.ico` בתיקיית השורש.
3. התקינו את התלויות הדרושות: `pip install -r requirements.txt`
4. לפקודת האריזה האופטימלית (בסביבה וירטואלית):
   `pyinstaller --onefile --windowed --icon=icon.ico ClassGadget.py`
